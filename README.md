# Java To-Do List Application

A simple Command Line To-Do List Application built using Java.

## Features

- Add new tasks
- Mark tasks as complete
- Remove tasks
- View all tasks

## Technologies Used

- Java
- OOP Concepts
- LinkedList Collection

## How It Works

The program runs in the terminal and provides a menu:

1. Add Task
2. Mark as Complete
3. Remove Task
4. View Tasks
5. Exit

Tasks are stored using a LinkedList and each task contains:

- Task Name
- Completion Status

## Example

Welcome to To-Do List Application

1 -> Add Task
2 -> Mark as complete
3 -> Remove Task
4 -> View Task
5 -> Exit
